Τι είναι η Τεχνητή Νοημοσύνη(ΑΙ);

Για χιλίαδες χρόνια προσπαθούμε να κατανοήσουμε το πως σκεπτόμαστε δηλαδή, πώς μια χούφτα ύλης μπορεί να αντιλαμβάνεται, να κατανοεί, να προβλέπτει και να χειρίζεται έναν κόσμο πολύ μεγαλύτερο και πολύ πιο πολύπλοκο απο τον εαυτό της. Η επιστήμη της τεχνητής νοημοσύνης προσπαθεί όχι μόνο να κατανοήσει πως δουλεύει ο ανθρώπινος νους άλλα παράλληλα να κατασκευάσει νοήμονες οντότητες. Η τεχνητή νοημοσύνη καθιστά τις μηχανές ικανές να μαθαίνουν δίνοντας τους ένα σύνολο δεδομένων εκπαίδευσης. Η μηχανή είναι σαν να παίρνει εμπειρία και αν την βάλουμε να τρέξει με νέα δεδομένα διαφορετικά απο τα δεδομένα εκπαίδευσης θα μπορέσει να προσαρμοστεί στα νέα δεδομένα και να εκτελεί εργασίες που θα εκτελούσε και ένας άνθρωπος.
Στις μέρες μας η τεχνητή νοημοσύνη εφαρμόζεται σε πολλούς τομείς όπως στην υγεία, στο εμπόρειο, στην παραγωγή, στον αθλητισμό, στο gaming και σε πάρα πολλούς άλλους τομείς. Σε κάθε κλάδο δηλαδή υπάρχει ζήτηση για τις δυνατότητες της τεχνητής νοημοσύνης. Ένα παράδειγμα είναι ότι εταιρείες σαν την Hive ή την Lime έχουν βάσεις δεδομένων απο τις οποίες εξάγουν κάποια δεδομένα για την συμπεριφορά των χρηστών της. Τα δεδομένα αυτά τα δίνουν σε αλγορίθμους τεχνητής νοημοσύνης ώστε να μπορούν να προβλέψουν πόσα ηλεκτρικά πατήνια χρειάζονται σε μια περιοχή όπως για παράδειγμα το Μοναστιράκι. Ένα άλλο παράδειγμα είναι ότι το YouTube κάνει χρήση αλγορίθμων τεχνητής νοημοσύνης για να μας προτείνει κάποια βίντεο.
Έτσι σύμφωνα με τα παραπάνω βλέπουμε ότι η τεχνητή νοημοσύνη έιναι αρκετά σημαντική για την κοινωνία. Γενικά βλέπουμε οτι η τεχνητή νοημοσύνη αναλύει περισσότερα και βαθύτερα τα δεδομένα που έχει, επιτυγχάνοντας απίστευτη ακρίβεια στις περισσότερες περιπτώσεις. Προσθέτει ευφυΐα στα υπάρχοντα συστήματα και προϊόντα και αξιοποιέι στο έπακρο τα δεδομένα με αποτέλεσμα η ποιότητα τους να βελτιώνεται συνέχεια.
Αξιοσημείωτο έιναι να αναφέρουμε μεγάλες εταιρείες που αξιοποιούν τα οφέλει της τεχνητής νοημοσύνης. Αρχικά έχουμε την ιστοσελίδα Alibaba η οποία βάζει τους αλγορίθμους τεχνητής νοημοσύνης για να προβλέψει προιόντα που πιθανόν να ενδιαφέρουν τους καταναλωτές της. Μία ακομή εταιρεία είναι η Alphabet η οποία ανήκει στην Google. Σκοπός αυτής της εταιρείας είναι να φτίαξει αμάξια που δεν χρειάζονται οδήγο(self-driving cars). Τα αυτοκίνητα κάνουν χρήση αλγορίθμων τεχνητής νοημοσύνης για να αναγνωρίσει αντικείμενα στο δρόμο, να προσαρμόζεται σε καταστάσεις κάθε φορά διαφορετικές μεταξύ τους. Γενικά υπάρχουν πάρα πολλά παραδείγματα εταιρείών που κάνουν χρήση της τεχνητής νοημοσύνης (Amazon, Apple, Baidy, Facebook, IBM, Microsoft).
Έχοντας πει λοιπόν όλα αυτά το μόνο σίγουρο είναι οτι η τεχνητή νοημοσύνη μπαίνει όλοενα και περισσότερο στην ζωή και καθημερηνότητα μας, βελτιώνει την ποιότητα της ζωής μας. Μας διευκολύνει σε διεργασίες που απαίτουσαν πολύ χρόνο απο τον άνθρωπο, δίνοντας του αρκέτα αξιόπιστα αποτελέσματα σε πολύ σύντομο χρονικό διάστημα.

Ιστορία τεχνητής νοημοσύνης

Κατά τη δεκαετία του 1940 εμφανίστηκε η πρώτη μαθηματική περιγραφή τεχνητού νευρωνικού δικτύου, με πολύ περιορισμένες δυνατότητες επίλυσης αριθμητικών προβλημάτων. Καθώς ήταν εμφανές ότι οι ηλεκτρονικές υπολογιστικές συσκευές που κατασκευάστηκαν μετά τον Β' Παγκόσμιο Πόλεμο ήταν ένα τελείως διαφορετικό είδος μηχανής από ό,τι προηγήθηκε, η συζήτηση για την πιθανότητα εμφάνισης μηχανών με νόηση ήταν στην ακμή της. Το 1950 ο μαθηματικός Άλαν Τούρινγκ, πατέρας της θεωρίας υπολογισμού και προπάτορας της τεχνητής νοημοσύνης, πρότεινε τη δοκιμή Τούρινγκ· μία απλή δοκιμασία που θα μπορούσε να εξακριβώσει αν μία μηχανή διαθέτει ευφυΐα. Η τεχνητή νοημοσύνη θεμελιώθηκε τυπικά ως πεδίο στη συνάντηση ορισμένων επιφανών Αμερικανών επιστημόνων του τομέα το 1956 (Τζον Μακάρθι, Μάρβιν Μίνσκυ, Κλοντ Σάνον κλπ).

Τομείς AI

Φροντίδα υγείας

Οι εφαρμογές του AI μπορούν να παρέχουν εξατομικευμένη διαχείριση φαρμάκων και αναγνώσεις ακτινογραφιών. Οι προσωπικοί βοηθοί φροντίδας υγείας μπορούν να ενεργήσουν ως σύμβουλοι διαβίωσης, να σας υπενθυμίζουν να παίρνετε τα φάρμακά σας, να ασκείστε ή να τρώτε πιο υγιεινά.

Παραγωγή

Το AI μπορεί να αναλύει εργοστασιακά δεδομένα μέσα από εφαρμογές διαδικτύου των πραγμάτων (ΙοΤ), καθώς τα δεδομένα ρέουν από τον διασυνδεδεμένο εξοπλισμό για να προβλέψουν το αναμενόμενο φορτίο ζήτησης με τη χρήση recurrent neural networks, ενός τύπου deep learning network που χρησιμοποιείται για ακολουθίες δεδομένων.

Αθλητισμός

Η τεχνητή νοημοσύνη χρησιμοποιείται για τη σύλληψη εικόνων αθλητικών παιχνιδιών και για να παρέχει στους προπονητές reports σχετικά με τον τρόπο καλύτερης οργάνωσης του παιχνιδιού, συμπεριλαμβανομένης της καλύτερης τοποθέτησης των παιχτών στο γήπεδο και της στρατηγικής.
