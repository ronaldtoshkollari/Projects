Η βάση είναι online οπότε μπορείτε να κάνετε χρήση του site χωρίς να κάνετε κάποιο restore της βάσης. Αν θέλετε όμως να
μπει τοπική βάση δεδομένων πρέπει να αλλάξουν το application properties στο spring boot project και να μπει σαν
endpoint το localhost, να μπει το port που ακούει η βάση καθώς και το όνομα της βάσης και τέλος το username/password
της βάσης. 
Μην δώσετε σημασία στο φάκελο .git το είχαμε για χρήση του git.
Καλό θα ήταν να το τρέξει απο το intellij, διότι εκεί το δουλεύαμε.
Ο φάκελος video-club περιέχει όλα τα αρχεία της εργασίας, το intellij απλά θα πρέπει να ανοίξει το φάκελο Movies το οποίο
είναι το spring boot project και όχι το φάκελο video-club.
